## How Each Agent Uses Real Data

### From AGENT_vs_AUTOMATION.md & GOOGLE_API_INTEGRATION.md

#### Destination Recommender Agent
```
Google API Data → LLM Reasoning → Personalization → Ranking
- Queries: search_places(), get_place_details()
- Returns: Real ratings, reviews, pricing
- LLM reasons about which destinations match
- Personalization applies user weights
- Result: Personalized recommendation with reasoning
```

#### Itinerary Planner Agent
```
Google API Data → Activity Selection → Routing → LLM Description
- Queries: search_places(), get_distances()
- Returns: Real attractions, travel times, opening hours
- Plans optimal sequence using real distances
- LLM creates personalized day descriptions
- Result: Detailed itinerary with real activity info
```

#### Budget Optimizer Agent
```
Google API Data → Cost Calculation → Optimization → LLM Advice
- Queries: search_places() for hotels/restaurants, get_distances()
- Returns: Real pricing, availability, transport costs
- LLM reasons about cost-benefit tradeoffs
- Personalizes based on user spending history
- Result: Optimized budget with alternatives
```
